# schwab 美股券商嘉信入金

## 电汇入金
### 嘉信美国账户电汇入金

如果有美国的银行卡，可以ach入金，ach入金可以免所有费用，没有的话只能电汇入金或者绑定wise入金

电汇入金的话，查看汇款方式

{{< image src="../pic/schwab-openaccount-07.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}


Account Number to Provide  这个后面8位就是 schwab 的账户ID

然后就可以在银行端填写上述信息，发起电汇了

## 添加额外账户入金

### 打开添加额外账户的页面

点击 add account
{{< image src="../pic/schwab-transferin-001.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}

点击continue继续
{{< image src="../pic/schwab-transferin-002.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}

如下图，选择 checking 和 yes

获取账号，填到下方
{{< image src="../pic/schwab-transferin-003.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}

滑到页面最下方，勾选方框后，点击 submit
{{< image src="../pic/schwab-transferin-004.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}

但是有一部分用户提交绑定后，会出现这个提示“此功能暂时不可用，请稍后再试。”，不管怎么尝试都会提示，

拨打客服电话反馈的说法也是不一样的，最后还是解决不了问题。其实只需要和客服要一个Moneylink的申请表，填完之后上传一下就好了


首先拨打嘉信中文客服电话400-120-6217，反馈说ACH账户无法绑定，需要他们发一份MoneyLink的申请表，有些客服会给你发，但有些客服不会。
需要坚持他们发一份给你就好了

具体怎么填，需要问客服，填好后，在如下位置上传

{{< image src="../pic/schwab-transferin-005.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}


### 添加额外账户绑定ibkr ach账户入金


{{< image src="../pic/schwab-transferin-ib-001.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}

获取路由账号 和 账户号码
{{< image src="../pic/schwab-transferin-ib-002.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}

绑定后，schwab 会向账号转入两笔金额低于1美元的汇款，需要把金额填到schwab，

并把接收到金额汇入schwab

然后就可以发起转账了
{{< image src="../pic/schwab-transferin-006.png" alt="schwab-ach-01" title="schwab-ach-01r" loading="lazy" >}}

> 注意 只建议从ibkr汇入schwab, 不建议 schwab 汇入 ibkr, 从schwab汇入ibkr可能导致ibkr封号
