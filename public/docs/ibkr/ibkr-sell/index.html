    ibkr盈透证券美股-股票买卖#

    美股除了期权可以做多，做空外，股票也可以做空，即在没有持仓的情况下，卖出股票或者卖出高于持仓股数的股票。

    股票交易中的买入和卖空是两种基本的操作方式，分别用于不同的市场预期和策略。以下是它们的详细介绍：

    1. 买入（做多）#

    买入股票是最常见的交易方式，投资者预期股票价格会上涨，通过低买高卖获利。

    操作流程：#

    1. 选择股票：通过研究公司基本面、技术分析或市场趋势，选择你认为有上涨潜力的股票。
    2. 下单买入：
      • 市价单：以当前市场价格立即买入。
      • 限价单：设定一个价格，只有当股票达到或低于该价格时才会买入。
    3. 持有股票：买入后，股票会进入你的持仓，等待价格上涨。
    4. 卖出平仓：当股票价格上涨到目标价位时，卖出股票实现盈利。

    特点：#

    • 风险有限：最大亏损为买入金额（如果股票价格跌至零）。
    • 收益潜力：理论上收益无限，取决于股票上涨空间。
    • 适合场景：看好某只股票或市场整体上涨时使用。

    2. 卖空（做空）#

    卖空是投资者预期股票价格会下跌时采取的策略，通过借入股票卖出，等待价格下跌后买回，赚取差价。

    操作流程：#

    1. 选择股票：通过分析认为某只股票价格会下跌。
    2. 借入股票：
      • 通过券商（如IBKR）借入股票（需支付借股费用）。
      • 确认股票可借（某些股票可能难以借到）。
    3. 下单卖出：
      • 市价单：以当前市场价格立即卖出。
      • 限价单：设定一个价格，只有当股票达到或高于该价格时才会卖出。
    4. 等待价格下跌：卖出后，等待股票价格下跌。
    5. 买回平仓：当股票价格下跌到目标价位时，买回股票归还给券商，赚取差价。

    特点：#

    • 风险较高：理论上亏损无限（如果股票价格上涨）。
    • 收益有限：最大收益为股票价格跌至零时的差价。
    • 适合场景：看空某只股票或市场整体下跌时使用。

    买入 vs 卖空 对比#

    特点 买入（做多） 卖空（做空）
    市场预期 预期股票价格上涨 预期股票价格下跌
    操作方式 买入股票并持有 借入股票卖出，等待下跌后买回
    风险 有限（最大亏损为买入金额） 无限（股价可能无限上涨）
    收益潜力 无限（股价可能无限上涨） 有限（最大收益为股价跌至零）
    费用 交易佣金 交易佣金 + 借股费用
    适合场景 牛市或看好某只股票 熊市或看空某只股票

    IBKR 卖空股票 （Shorting Stocks）#

    IBKR 账户不能同时持有同一证券的多头和空头头寸，也称为“反向卖空”（shorting against the box）。

    因此，没有买多和卖空的订单类型 - 只有买入或卖出。
