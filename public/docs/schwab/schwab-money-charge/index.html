    电汇入金#

    嘉信美国账户电汇入金#

    如果有美国的银行卡，可以ach入金，ach入金可以免所有费用，没有的话只能电汇入金或者绑定wise入金

    电汇入金的话，查看汇款方式

    Account Number to Provide 这个后面8位就是 schwab 的账户ID

    然后就可以在银行端填写上述信息，发起电汇了

    嘉信香港账户电汇入金#

    获取嘉信香港账号#

    可以看到香港的账户如图

    或者打开 https://international.schwab.com/resource/instructions-to-transfer-money-to-your-schwab-account

    Bank Name Citibank N.A., Hong Kong
    Bank Address 3 Garden Road, Central, Hong Kong
    SWIFT ID CITIHKHXXXX
    Bank Code 006
    Branch Coder 391
    Schwab Account Number (add branch coder in front of account number for example: 39162286838)
    Schwab Account Name Charles Schwab & Co., Inc.
    Schwab Account Address 3000 Schwab Way, Westlake, TX 76262, USA
    For Further Credit To 请在汇款指示的备注栏中填写：Schwab 账户号（8 位） + 账户登记姓名（与 Schwab 账户一致）

    注意嘉信理财使用的是香港 Citibank 的户口。

    打开任意一个香港银行的手机应用，然后添加收款人，发起转账，可以先小额金额转账验证是否能正确到账

    添加额外账户入金#

    打开添加额外账户的页面#

    点击 add account

    点击continue继续

    如下图，选择 checking 和 yes

    获取账号，填到下方

    滑到页面最下方，勾选方框后，点击 submit

    但是有一部分用户提交绑定后，会出现这个提示“此功能暂时不可用，请稍后再试。”，不管怎么尝试都会提示，

    拨打客服电话反馈的说法也是不一样的，最后还是解决不了问题。其实只需要和客服要一个Moneylink的申请表，填完之后上传一下就好了

    首先拨打嘉信中文客服电话400-120-6217，反馈说ACH账户无法绑定，需要他们发一份MoneyLink的申请表，有些客服会给你发，但有些客服不会。 需要坚持他们发一份给你就好了

    具体怎么填，需要问客服，填好后，在如下位置上传

    添加额外账户绑定ibkr ach账户入金#

    获取路由账号 和 账户号码

    绑定后，schwab 会向账号转入两笔金额低于1美元的汇款，需要把金额填到schwab，

    并把接收到金额汇入schwab

    然后就可以发起转账了

    注意 只建议从ibkr汇入schwab, 不建议 schwab 汇入 ibkr, 从schwab汇入ibkr可能导致ibkr封号
